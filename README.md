1. **Home (index.html)** - Landing page with hero section, featured events, statistics, and more

## 🚀 Getting Started

### View Locally
1. Clone this repository
   ```bash
   git clone https://github.com/yourusername/University-Events.git
   ```
2. Navigate to the project folder
   ```bash
   cd University-Events
   ```
3. Open `index.html` in your browser

### Deploy to GitHub Pages
1. Push all files to GitHub
2. Go to Settings → Pages
3. Select main branch
4. Save and wait for deployment
5. Access your site at: `https://yourusername.github.io/University-Events/`
